Тестовое задание должен быть выполнено на языке программирования C#(.NET Framework) и использованием для хранения информации базы данных PostgreSQL.
	
Задача:
Реализовать загрузчик закупки из разных источников.
Загрузчик должен иметь возможность загружать информацию  из двух форматов источников: ссылка с сайта, XML.
Итогом работы загрузчика, для всех источников, должны быть объект одного и того же класса и поддерживаться загрузка полученных данных в БД.

Данные, которые необходимо получить:
	Общая информация о закупке - Номер закупки, наименование закупки, начальная цена, дата публикации;
	Заказчики - Название организации, ИНН (если указан);

Форматы и источники загрузки:
	Ссылка - https://tenmon.ru/1/0123200000319002908
	XML - 0123200000319002908.xml

Результат тестового задания:
	Исходные данные проекта на языке C#
	Запросы для создания структуры базы данных (возможен дамп базы данных в формате .sql)
	Список подключаемых библиотек (если использовались)
